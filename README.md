## About
Team DEVROOTS is building a web portfolio for the PreserveMy.World (PMW) Internship 2026 program, under the Platform & Web Engineering track. The site presents the team, the members, and the PMW mission in one page.

## What this repo contains
- `index.html` — the live team page (hero section, team member grid, PMW description, footer)
- `image1.jpg` — team logo used on the page
- `README.md` — this file, describing the team and repo

## Mission
Team DEVROOTS builds web systems that help preserve and present cultural heritage, as part of PreserveMy.World's Platform & Web Engineering track.

## Tech used
- HTML, CSS, JavaScript (single-file site, no framework)
- Encrypted-text hero effect, comet hover effect on team cards
- Hosted on GitHub Pages
